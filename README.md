# Fishing Game (C#)

## Description
Card game developed in C# using object-oriented programming (OOP) and strategy patterns.

## Features
- Game logic and rules implementation
- Player management
- Multiple strategies via `IStrategy` (Basic, BlockOpponent, MinimizePoints)
- Clean OOP structure (classes and game board)

## Tech Stack
- C#
- .NET (project file: `FishingGame.csproj`)

## How to Run
1. Open the project in Visual Studio / Rider
2. Build and run `Program.cs`
